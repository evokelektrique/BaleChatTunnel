# Flutter App Icon Pack

This pack contains release-ready app icon exports generated from the approved dark cube + mint logo design.

## Included
- Android launcher icons: mdpi, hdpi, xhdpi, xxhdpi, xxxhdpi
- Android round launcher icons
- Android adaptive icon foregrounds and XML templates
- Google Play 512px icon
- Flutter Web PWA icons and favicon
- Windows PNG icons and multi-size .ico
- macOS `.iconset` PNGs
- Linux hicolor app icons
- 1024px and 512px source PNGs
